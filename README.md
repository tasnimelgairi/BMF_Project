# Masterfile App

Projet PFE pour le traitement des fichiers EQT/PRC.

## Structure
- frontend : interface utilisateur
- backend : API et traitement des données
- data : fichiers d’entrée et de sortie
- docs : documentation du projet

## Stack technique
- Frontend : React + Vite
- Backend : FastAPI
- Traitement : Pandas, NumPy
- Qualité : Great Expectations
- Base locale : DuckDB"# BMF_Project"
